This projects implements a package dependecy resolver
See task.pdf for more details


To run makefile use:
"make -f makefile"

To run tests use:
chmod 777 "tests.sh"
./tests.sh

To use with customs parameters use:
./dependency_resolver "filename" "packageName" 
(packageName must consist of a single letter!)
